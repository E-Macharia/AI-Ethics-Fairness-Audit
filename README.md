## Team Members

## Project Overview
This repository contains our submission for the AI Ethics assignment focused on analyzing and mitigating biases in AI systems. The project includes theoretical analysis, case studies, and practical implementation of fairness metrics.

## Repository Structure
```
AI_Ethics_assignment/
├── README.md                   # This file
├── theoretical_analysis.pdf    # Answers to Part 1 & 2
├── code/
│   ├── requirements.txt        # Dependencies
│   ├── fairness_audit.ipynb    # Jupyter notebook for Part 3
│   └── visualizations/         # Generated plots and charts
├── data/                       # Dataset storage
└── bonus/
    └── healthcare_ai_policy.md # Bonus task: Healthcare AI Policy
```

## Setup Instructions
1. Clone this repository
2. Install dependencies: `pip install -r code/requirements.txt`
3. Open `code/fairness_audit.ipynb` to view the analysis

## Submission Components
- Part 1 & 2: Theoretical understanding and case studies (PDF)
- Part 3: Technical audit (Jupyter notebook + visualizations)
- Part 4: Ethical reflection (included in theoretical_analysis.pdf)
- Bonus: Healthcare AI policy proposal (markdown file)
